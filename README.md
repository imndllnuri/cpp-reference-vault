# C++ Mastery Reference

A complete personal reference for C++ — covering everything from compiler toolchains and language fundamentals to templates, concurrency, systems programming, and algorithms. Built so that nothing needs to be looked up elsewhere.

## Parts

| File                                         | Description                                                           |
| -------------------------------------------- | --------------------------------------------------------------------- |
| [[Part 0 — Environment & Toolchain]]         | Compilers, build systems, debuggers, package managers, sanitizers     |
| [[Part 1 — C++ Fundamentals]]                | History, the compilation model, types, literals, type qualifiers      |
| [[Part 2 — Object-Oriented Programming]]     | Classes, constructors, inheritance, polymorphism, the rule of five    |
| [[Part 3 — Memory Management]]               | Stack vs heap, RAII, smart pointers, memory errors, custom allocators |
| [[Part 4 — Standard Template Library (STL)]] | Containers, iterators, algorithms, string, span, ranges               |
| [[Part 5 — Modern C++ (C++11 to C++23)]]     | Move semantics, lambdas, constexpr, concepts, modules, coroutines     |
| [[Part 6 — Templates & Metaprogramming]]     | Function/class templates, specialization, SFINAE, type traits, CRTP   |
| [[Part 7 — Concurrency & Multithreading]]    | Threads, mutexes, atomics, futures, coroutines, common pitfalls       |
| [[Part 8 — Systems Programming]]             | File I/O, POSIX, processes, signals, IPC, sockets, dynamic linking    |
| [[Part 9 — Design Patterns]]                 | SOLID, GoF patterns, C++ idioms: PIMPL, CRTP, type erasure, NVI       |
| [[Part 10 — Data Structures & Algorithms]]   | Complexity, data structures, sorting, searching, DP, graphs           |
